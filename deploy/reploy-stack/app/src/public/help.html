📚 Getting Started

What is NyLi Assets? ▼

NyLi Assets is a software license auditing platform that helps organizations track software usage, compare user lists, identify compliance issues, and communicate with users about their software access.

How do I create my first project? ▼

Navigate to Projects from the top navigation bar. Click the New Project button and fill in:

  • Project Name — a descriptive name for this audit
  • Client — the organization being audited
  • Product Name — the software product under review

Once created, you can upload user lists and run comparisons from the project detail page.

How do I upload user data? ▼

From a project detail page, use the Upload section to import CSV or Excel files containing your user lists. You'll typically need two lists:

  • Entitled Users — users who should have access
  • Active Users — users who actually have access

NyLi Assets will compare these lists and flag discrepancies.

What file formats are supported? ▼

NyLi Assets accepts CSV (.csv) and Excel (.xlsx, .xls) files. Ensure your files have a header row with column names so the system can identify the data fields.

📂 Projects & Comparisons

How does the comparison work? ▼

When you run a comparison, NyLi Assets matches users from your entitled list against your active list. It identifies:

  • Flagged users — active but not entitled (potential compliance issues)
  • Missing users — entitled but not active
  • Matched users — present in both lists

Results are displayed on the project detail page and tracked in the dashboard.

Can I run AI-powered analysis? ▼

Yes! If your organization has the AI analysis feature enabled, you can run an automated analysis on comparison results. The AI reviews flagged users and provides insights and recommendations to help prioritize your audit actions.

AI analysis results appear in the Analysis section of the project detail page.

How do I archive or delete a project? ▼

From the Projects page, locate the project you want to manage. Use the project actions menu to change the status to Archived or to delete it. Archived projects can be restored later; deleted projects are permanently removed.

📧 Communications

How do I send emails to flagged users? ▼

Navigate to Communications from the top menu. Select the project and choose the users you want to contact. You can use email templates or compose a custom message. NyLi Assets tracks all sent communications and user responses.

How do I track email responses? ▼

The Communications page shows the status of all sent emails. Users who receive audit emails can confirm or revoke their software access through a secure link. Their responses are tracked automatically and reflected in your project data.

📊 Dashboard & History

What does the dashboard show? ▼

The Dashboard provides an at-a-glance overview of your audit activity, including:

  • Total projects, flagged users, and response rates
  • Charts showing trends over time
  • Recent comparison results
  • Cost-per-user financial insights
Where can I view past audit runs? ▼

The History page (accessible from the dashboard) displays all past comparison runs with timestamps, flagged counts, and trend charts. You can filter by project and date range.

⚙ Account & Settings

How do I change my password? ▼

Go to Account Settings from the user menu dropdown. In the Security section, enter your current password and your new password, then click Update Password.

How do I set my preferences? ▼

Go to Preferences from the user menu or navigate to Settings > Preferences. You can configure:

  • Theme — Light or Dark mode
  • Language — Interface language
  • Notifications — Email notification preferences
What user roles are available? ▼

NyLi Assets has several roles:

  • Admin — Full access to all features and user management
  • Manager — Can manage projects, run comparisons, and send communications
  • Viewer — Read-only access to dashboards and reports

Your role is assigned by your organization's administrator.

📅 Calendar

What is the Calendar for? ▼

The Calendar helps you schedule and track audit deadlines, review dates, and follow-up reminders. Events are color-coded by type and can be filtered by project.

How do I add a calendar event? ▼

Click on any date in the calendar to create a new event. Fill in the event title, description, and optionally link it to a project. Events can be edited or deleted at any time.
