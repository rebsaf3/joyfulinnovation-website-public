InsightPilot

InsightPilot helps teams turn raw business data into interactive dashboards in minutes. Upload CSV, Excel, or JSON, describe what you want in plain English, and generate production quality Plotly charts without writing code.

Product Suite

InsightPilot completes the Joyful Innovation suite with AssetPilot and FlowPilot.

Why Teams Choose InsightPilot

Most organizations wait on analyst queues for one off reporting, and traditional BI tools can be slow to adopt. InsightPilot gives teams a faster path from data to decision with plain language analysis, reusable workflows, and production ready security controls.

  • Reduce time to insight with natural language prompts that generate interactive Plotly charts in seconds
  • Enable non technical teams to self serve analytics with no SQL, Python, or drag and drop builder
  • Standardize reporting with per project instructions, prompt templates, and chart revision workflows
  • Support collaboration with role based access in secure multi tenant workspaces
  • Meet enterprise requirements with SSO, 2FA, audit logging, API key controls, and white label options

How InsightPilot Works

1. Upload Data

Import CSV, Excel, or JSON files. InsightPilot automatically profiles columns, types, null counts, and summary stats.

2. Describe the Goal

Ask for the chart you need in plain English, such as revenue trends or category breakdowns.

3. Review Output

Claude generates Plotly code and an interactive chart. If code fails, InsightPilot can auto retry with error feedback.

4. Refine and Save

Request revisions, apply project instructions for consistency, and save final charts to dashboards.

5. Share and Scale

Collaborate through workspace roles, export dashboards on paid tiers, and add API access for automated workflows.

Data Pipeline

Upload CSV, Excel, or JSON with smart parsing, encoding detection, and automated column profiling for cleaner analysis inputs.

AI Analysis Flow

Use the three step workflow of Describe, Review, and Save. Revise charts with follow up prompts and keep full code transparency.

Dashboard Delivery

Save multi chart dashboards, manage chart order, and export multi page PDF reports on Pro and Enterprise plans.

Security and Governance

Password auth, Google and Microsoft SSO, SAML, TOTP and email 2FA, backup codes, API key controls, and full admin audit trails.

Billing and Usage

Credit based pricing maps usage to AI consumption, with monthly allocations, usage dashboards, and ledger level tracking.

API and Automation

Optional REST API add on (Pro and Enterprise) provides endpoint access for projects, files, dashboards, analysis, and workspace workflows.

Plan Snapshot

Free - $0/mo

Best for initial evaluation and light individual use.

  • 50 credits per month
  • 1 member, 1 daily upload, 10 MB file size
  • Up to 3 dashboards
  • No chart revisions, no export, no top ups

Pro - $29/mo

Best for teams that need recurring reporting and collaboration.

  • 500 credits per month
  • Up to 10 members, 10 daily uploads, 100 MB files
  • Up to 25 dashboards and unlimited chart revisions
  • PDF/PNG export, Google and Microsoft SSO

Enterprise - $99/mo

Best for larger organizations with advanced security and branding requirements.

  • 2,000 credits per month
  • Unlimited members, uploads, and dashboards
  • 500 MB max file size, unlimited revisions, PDF/PNG export
  • Google/Microsoft/SAML SSO, required 2FA, full white label

API access is available as a $15/mo add on for Pro and Enterprise. Credit top ups are available on Pro and Enterprise at $10 per 100 credits.
