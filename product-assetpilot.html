NyLi Assets

NyLi Assets helps teams move from raw license and directory exports to verified reclaim decisions and evidence they can defend. Upload source files, run structured comparisons, track verification responses, and export audit-ready records.

Product Suite Fit

NyLi Assets is the governance and detection layer in the Joyful Innovation suite. Teams pair NyLi Assets with NyLi Insights to turn audit and reclaim evidence into leadership-ready reporting.

How NyLi Assets Works

From raw exports to verified license reclamation in four steps

1. Upload source files

Upload software license assignments and Active Directory user records as CSV files. NyLi Assets validates file type and size before processing (10 MB upload limit).

2. Run comparison

Run the comparison to generate matched users, license-only users, AD-only users, and field-level differences across matched records.

3. Send verification

Send one-click verification emails to flagged users and track pending, confirmed, and revoked responses as auditable evidence.

4. Reclaim and document

Build a reclaim list and export timestamped, IP-logged evidence for procurement and compliance review before final actions are taken.

Built for operational trust

Every major action is logged with timestamps, user context, and audit detail so teams can move fast without sacrificing governance.

Core Capabilities

Audit engine

Dual CSV comparison with drag-and-drop upload, key-column detection across common schemas, and detailed discrepancy reporting by user and field.

Verification workflow

Tokenized email verification with Yes/No response tracking, a 14-day non-response warning, and clear response history.

Project lifecycle

Organize audits by software product, manage project status (Active, Completed, Archived), and customize project-level email templates.

Analytics and AI analysis

Track KPIs and trends, then use AI-generated summaries, anomaly flags, and user categories to prioritize review and support human decisions.

Security and tenancy

Company-scoped multi-tenant isolation, role-based access (member/admin/superadmin), rate limiting, and session controls.

Admin and billing controls

Stripe-backed subscriptions, invoice tracking, bulk user administration, and audit log export support up to 10,000 rows.
