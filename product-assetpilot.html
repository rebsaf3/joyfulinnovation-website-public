AssetPilot

AssetPilot is software license intelligence for teams that need to reduce waste quickly and defend every reclaim decision. Upload license and directory data, run automated comparisons, trigger user verification, and export evidence-ready audit records.

Product Suite Fit

AssetPilot is the software governance layer in the Joyful Innovation suite, and it connects cleanly with FlowPilot and InsightPilot initiatives.

How AssetPilot Works

From raw exports to verified license reclamation in four steps

1. Upload source files

Upload two CSV files: software license assignments and Active Directory user records. The system validates file type and size with a 10 MB upload limit.

2. Run comparison

AssetPilot auto-detects key columns and produces matched users, users only in license files, users only in AD, and field-level differences.

3. Send verification

Send one-click confirmation emails to flagged users with cryptographic tokens. Track pending, confirmed, and revoked responses with duplicate-response prevention.

4. Reclaim and document

Use dashboard trends, status analytics, and CSV exports to reclaim licenses and provide timestamped, IP-logged records for procurement and compliance review.

Built for operational trust

Every major action is logged with timestamps, user context, and audit detail so teams can move fast without sacrificing governance.

Core Capabilities

Audit engine

Dual CSV comparison with drag-and-drop upload, schema-tolerant key detection, and detailed discrepancy reporting across users and fields.

Verification workflow

Tokenized email verification with Yes/No response tracking, 14-day auto-revocation warning, and immutable response history.

Project lifecycle

Organize audits by software product, manage project status (Active, Completed, Archived), and customize project-level email templates.

Analytics and AI analysis

Track KPIs and trend charts, then generate executive summaries, anomaly flags, and user risk categories through Claude or OpenAI providers.

Security and tenancy

Company-scoped multi-tenant data isolation, role-based access (member/admin/superadmin), strong rate limits, and session controls.

Admin and billing controls

Stripe-backed subscriptions, invoice tracking, bulk user administration, and export-ready audit logs up to 10,000 rows.
