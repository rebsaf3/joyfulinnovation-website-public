How long does a first run take end to end?

Most teams can run the compare step in minutes once they export two CSV files. Verification runs on your schedule, then the reclaim list updates as confirmations arrive.

What do we need to export from our license system?

A CSV file with user identifier and license assignment is enough to start. More fields improve matching, but NyLi Assets can work with a minimal export.

Do you store our CSV data and for how long?

Uploaded CSV row data is stored in the application database to support comparisons, verification workflows, and audit history. Verification responses include timestamp and IP logging. Retention terms are defined in customer agreements and implementation settings. See our Privacy Policy for more detail.

Can we disable verification emails and only use compare?

Yes. Many teams start with compare only to identify obvious waste and validate data quality before turning on verification.

How do you prevent risky deactivations?

NyLi Assets provides recommendations, not automatic deactivations. It gives evidence, status tracking, and verification signals so people can make safe final decisions.

What security controls exist for multi tenant separation?

NyLi Assets applies tenant isolation at the application level so company data remains scoped and separated. Contact us for a detailed security review for your environment.

Do you support API access?

API access is available on Enterprise plans. Contact us to scope the right integration approach for your environment.

What does onboarding look like?

We guide your export setup, run the first compare cycle with you, and provide a walkthrough of verification and reclaim actions so your team can operate confidently.
