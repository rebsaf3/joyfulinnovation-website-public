    How long does a first run take end to end?

    Most teams run the compare step in minutes once two CSV files are ready. Verification then runs on your schedule, and reclaim candidates update as responses arrive.

    What do we need to export from our license system?

    A CSV with user identifier and license assignment is enough to start. Additional fields improve matching quality, but NyLi Assets works with minimal exports.

    Do you store our CSV data and for how long?

    Uploaded CSV row data is stored in the application database to support comparisons, verification workflows, and audit history. Verification responses include timestamp and IP logging. Retention terms are defined in customer agreements and implementation settings. See our Privacy Policy for more detail.

    Can we disable verification emails and only use compare?

    Yes. Compare-only is a common starting approach to identify obvious waste and validate data quality before enabling verification.

    How do you prevent risky deactivations?

    NyLi Assets provides recommendations, not automatic deactivations. It provides evidence, status tracking, and verification signals so human owners make final decisions.

    What security controls exist for multi tenant separation?

    NyLi Assets applies tenant isolation at the application level so company data remains scoped and separated. Contact us for a detailed security review for your environment.

    Do you support API access?

    API access is available on Enterprise plans. Use Support to scope the right integration approach for your environment.

    What does onboarding look like?

    We guide export setup, run the first compare cycle with you, and walk through verification and reclaim actions so your team can operate with confidence.
