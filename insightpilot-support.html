Quick Start

1. Create Workspace

Create your workspace, add team members, and assign roles for access control.

2. Upload Data

Upload CSV, Excel, or JSON files. InsightPilot profiles columns and prepares data for analysis.

3. Describe Analysis

Use plain English prompts such as Show revenue trends by category for chart generation.

4. Review and Revise

Inspect chart output, request modifications, and use follow up prompts for fast iteration.

5. Save Dashboard

Store charts in dashboards, reorder layouts, and prepare exports for stakeholders.

6. Operationalize

Enable project instructions, prompt templates, and API access for repeatable workflows.

Feature Documentation

Authentication and Access

Email and password authentication, Google and Microsoft OIDC, SAML support, TOTP and email 2FA, backup codes, and session management with remote revocation.

Data Pipeline

CSV, Excel, and JSON uploads with encoding detection, nested JSON normalization, and automated profiling for types, nulls, unique values, and statistical summaries.

AI Chart Generation

Plain language prompts to production Plotly charts, three step Describe Review Save workflow, project level instructions, prompt templates, and revision support.

Security Controls

Sandboxed execution with AST validation, restricted builtins, blocked attribute patterns, execution timeout, and read only data protections.

Dashboards and Export

Create and manage multi chart dashboards with chart ordering and PDF export support on Pro and Enterprise plans.

API and Automation

REST API endpoints for project, file, chart, and dashboard workflows with API keys, permissions, and rate limiting.

Troubleshooting

Upload issues

Check file format and size limits by plan. CSV encoding is auto detected, but malformed files may still require correction before upload.

Chart generation errors

InsightPilot uses automatic retry for generated code failures. If errors persist, simplify the prompt and validate key data fields in your file.

Access and permission issues

Confirm workspace role, invitation status, and SSO configuration. Enterprise SSO setup may require IdP metadata review.

Billing and credits

Review subscription tier, credit balance, and add on settings. AI analysis requests require available credits before execution.
